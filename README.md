# Upstate Liberty

## Installation
- install Ultimate ASI Loader (https://github.com/ThirteenAG/Ultimate-ASI-Loader)
- copy the mod files into the game directory
- supported exe versions: 1.0 only
- optionally install QOL mods (SilentPatch, Widescreen Fix, SkyGFX)

## Compability notes
- **SAVES ARE NOT COMPATIBLE!!!**
- **Modloader** does not support .asi and .zon files correctly (to be fixed in ML update).
- If you are using Upstate Liberty together with any of the following mods, make sure they are up to date:
  - Outdated versions of **SilentPatch** crash on the loading screen. Fixed in **SilentPatchIII Build 9.2** (or newer).
  - Outdated vesions of **gtadebug** cause a crash when starting a New Game in freeroam mode, and spawning vehicles is broken.
    Fixed in **gtadebug v1.4.1** (or newer).

## Freeroam setup
- rename main_d.scm to main.scm

Or
- install debug menu (https://github.com/aap/debugmenu)
- install gtadebug (rename iii_debug.dll to iii_debug.asi) (https://github.com/aap/gtadebug)
- hold 'D' while the game is loading to launch the game in freeroam mode

To teleport to various places in Upstate, use **Square** + **Triangle** + **DPad** (**Jump** + **Enter vehicle** + **Arrow keys** on the keyboard).
